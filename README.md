                          1. ВВЕДЕНИЕ
================================================================================

BudgetApp — веб-приложение для учёта личных доходов и расходов.
Позволяет регистрироваться, добавлять финансовые операции, прикреплять
чеки, просматривать историю и анализировать статистику.

Актуальность: в современных условиях человек совершает множество финансовых
операций ежедневно. Без систематического учёта сложно понять, куда уходят
деньги. Приложение решает эту проблему, предоставляя удобный инструмент
для контроля финансов, доступный с любого устройства через браузер.

Практическая значимость: приложение может использоваться для личного
финансового планирования, обучения основам бюджетирования, а также как
демонстрационный проект при изучении технологий веб-разработки на Python.

================================================================================
                  3. ФУНКЦИОНАЛЬНЫЕ ВОЗМОЖНОСТИ
================================================================================

ОСНОВНЫЕ ФУНКЦИИ:

- Регистрация пользователя (email + пароль)
- Авторизация (вход в личный кабинет)
- Добавление транзакции (доход или расход)
- Указание категории, суммы и описания
- Прикрепление чека (изображение или PDF)
- Просмотр истории всех операций
- Статистика: доходы, расходы, баланс
- Курсы валют (USD, EUR к RUB)
- Удаление транзакций с подтверждением
- REST API для программного доступа

ЧЕМ ПРИЛОЖЕНИЕ ПОМОГАЕТ ПОЛЬЗОВАТЕЛЮ:

- Контроль финансов: точные суммы доходов и расходов.
- Анализ трат: видно, на какие категории уходит больше денег.
- Планирование бюджета: баланс помогает планировать накопления.
- Хранение чеков: электронные копии не теряются.
- Экономия времени: добавление записи занимает меньше минуты.
- Доступность: приложение открывается с любого устройства.

================================================================================
                6. ОПИСАНИЕ ОСНОВНЫХ МОДУЛЕЙ
================================================================================

app.py — ГЛАВНЫЙ МОДУЛЬ

Содержит маршрутизацию и бизнес-логику. Инициализирует Flask,
подключает расширения (SQLAlchemy, LoginManager, RESTful API).

Маршруты:
/                               Главная страница (статистика и транзакции)
/register                       Регистрация пользователя
/login                          Вход в систему
/logout                         Выход из системы
/add                            Добавление транзакции
/transaction/<id>               Просмотр транзакции
/transaction/<id>/delete        Удаление транзакции (POST)

Декоратор @check_session проверяет соответствие user_id в сессии
и в current_user. При несовпадении сессия сбрасывается.

models.py — ORM-МОДЕЛИ

Содержит классы User и Transaction, описывающие таблицы БД.
Связь: один пользователь -> много транзакций.
Каскадное удаление: при удалении пользователя удаляются все его записи.

forms.py — ФОРМЫ

RegisterForm:   username, email, password (с валидацией)
LoginForm:      email, password
TransactionForm: amount, type, category, description, receipt (файл)

utils.py — ВСПОМОГАТЕЛЬНЫЕ ФУНКЦИИ

save_receipt()        Сохранение чека со сжатием через Pillow
get_currency_rates()  Получение курсов USD/EUR через exchangerate-api.com
calculate_totals()    Подсчёт доходов, расходов и баланса
